## 已完成的性能优化（弱 VPS 并发承载）

- SQLite 启用 WAL 模式，支持读写并发，`busy_timeout=5000` 避免写锁冲突。
- LLM 调用添加 `asyncio.Semaphore(5)` 并发控制，防止内存暴涨。
- Uvicorn `--limit-concurrency 30`，超限返回 503 而非 OOM。
- Docker 容器内存限制（backend 512M / frontend 64M）。
- Nginx gzip 压缩 + 静态资源长期缓存。
- Caddy 反代 60s 超时配置。
- 预期承载能力：1-2 核 / 1-2G 内存 VPS 上约 15-25 人同时提问。
