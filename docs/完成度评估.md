## 剩余可选优化

1. 将当前本地混合检索升级为真正的向量索引服务，例如 Chroma 或 FAISS，并保留现有阈值守门逻辑。
2. 把 PDF 索引重建改成后台任务，前端显示进度，避免大文件重建时阻塞请求。
3. 增加管理后台的批量导入/导出，方便老师验收时查看问答对和知识图谱数据。
4. 将长回答来源进一步按“核心证据”和“补充证据”分组，提高引用可读性。
5. 为 MiniMax 调用增加 token、耗时、失败率统计面板。
