# 高级统计方法智能问答系统

本上下文定义题目二项目中的核心领域语言，避免后续实现、数据构建和报告写作时混用概念。

## Language

**智能问答系统**:
面向高级统计方法课程学习的中文问答工具。
_Avoid_: 普通聊天机器人

**教材语料**:
用于检索增强生成的教材正文来源。
_Avoid_: 教材附件, PDF 图片

**知识点**:
统计学习中的一个可解释概念或方法节点。
_Avoid_: 关键词, 标签

**问答对**:
一条经过整理的课程问题、答案、类型和关联知识点记录。
_Avoid_: 聊天记录, 训练样本

**RAG 文本块**:
从教材语料中切分出的可检索正文片段。
_Avoid_: 段落缓存, 原文截图

**知识图谱**:
由知识点和知识点之间的课程关系组成的概念网络。
_Avoid_: 思维导图

**图谱子图**:
围绕一个或多个核心知识点展示的局部关系网络。
_Avoid_: 全量图谱截图

**引用来源**:
回答所依据的教材章节、页码或图谱证据。
_Avoid_: 参考资料泛称

**关系查询**:
询问两个或多个知识点之间区别、联系或路径的问题。
_Avoid_: 模糊比较

**应用场景问题**:
询问统计方法适用任务、限制条件或使用案例的问题。
_Avoid_: 案例闲聊

## Relationships

- 一个 **问答对** 关联一个或多个 **知识点**。
- 一个 **RAG 文本块** 来自一个 **教材语料**。
- 一个 **智能问答系统** 使用 **RAG 文本块** 和 **知识图谱** 生成回答。
- 一个 **关系查询** 应返回相关 **知识点** 之间的关系说明，并尽可能展示 **图谱子图**。
- 一个回答可以包含多个 **引用来源**。

## Example dialogue

> **Dev:** “用户问‘岭回归和 Lasso 有什么区别’，这是普通 RAG 问题吗？”
> **Domain expert:** “不是，它是 **关系查询**。回答需要教材证据，也要展示岭回归和 Lasso 在 **知识图谱** 中的关系。”

## Flagged ambiguities

- “教材”在本项目中不能直接等同于 **教材语料**；扫描版中文 PDF 目前只是图片型资料，不能直接作为 RAG 文本来源。
- “知识图谱”不是装饰性的可视化图片；它必须支持节点点击、关系查询、问题推荐和回答页子图展示。
