# 题目二目标契约

## 目标

完成“基于高级统计方法课程知识图谱的智能问答系统设计与实现”的完整课程项目。交付物必须包含可运行源码、中文 Web 界面、课程数据、RAG 问答、知识图谱辅助问答、用户管理、运行说明和实验报告。

## 不可降级要求

- 使用 `ISLRv2_corrected_June_2023.pdf` 作为 RAG 主语料。
- 系统界面、问答对、回答内容、README 和实验报告均使用简体中文。
- 不把真实 API Key 写入代码、报告或示例文件。
- 不默认 OCR 中文扫描教材。
- 不依赖 Neo4j 才能运行；知识图谱必须能本地开箱演示。

## 技术默认

- 后端：FastAPI、SQLite、SQLAlchemy。
- 检索：优先 Chroma + sentence-transformers；不可用时自动降级到本地混合检索。
- 图谱：SQLite 存储 + NetworkX/内置图查询逻辑。
- 前端：React、TypeScript、Vite。
- LLM：MiniMax OpenAI-compatible Chat Completions。

## 完成定义

- 至少 60 个中文问答对、60 个知识点、100 条图谱边。
- 至少 200 个教材文本块，带章节、页码、来源文件元数据。
- 注册、登录、问答、来源引用、图谱子图、节点推荐、问答对 CRUD、历史记录均可用。
- 后端测试和端到端烟测通过。
- 本地服务实际启动过，并在 README 中记录运行命令。
- 实验报告覆盖需求、架构、数据、RAG、知识图谱、测试与效果分析。
